pyfivbvis

Python client for FIVB VIS web service system - openvolley

Contents

Welcome!

This library provides a comprehensive interface to interact with the FIVB VIS web service system.

If you need to loop through some of the functions using threading might be an advantage in order to speed up the process.

Offical documentation for the FIVB VisWebService: https://www.fivb.org/VisSDK/VisWebService/#Introduction.html

Installation

You can install your package using pip:

pip install pyfivbvis

When you need to upgrade:

pip install pyfivbvis -U

Beach Tournament List

https://www.fivb.org/VisSDK/VisWebService/#GetBeachTournament.html

Using this function will return every beach tournament in the FIVB VIS database. This is adventagous when you need to use all tournament numbers in order to process further data.

                                 
from pyfivbvis.vis_fetcher import FivbVis as vis
all_tournaments = vis.fetch_beach_tournament_list()
print(all_tournaments)
